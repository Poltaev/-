Приложение для просмотра подробных показателей погоды
Использует https://openweathermap.org/forecast5 , OpenWeather Api
Интерфейс это - три экрана которые показывают погоду на ближайшие 3 часа,
на ближайшие сутки с шагом в 3 часа и на ближайшие 5 дней с шагом в 12 часов. 
Для реализации использовал прокручивающиеся списки.
